# ⚡ Electricity Explorer — Singapore O-Level Physics

An interactive web application for teaching and learning **Current Electricity**, **DC Circuits**, and **Practical Electricity**, aligned to the **Singapore-Cambridge GCE O-Level Physics** syllabus.

## 🎯 Features

### 📘 Chapter 1: Current Electricity
| Topic | Interactive Elements |
|---|---|
| **Electric Current** (I = Q/t) | Animated electron flow with speed control; current calculator |
| **Potential Difference** (V = W/Q) | Voltmeter circuit diagram; water analogy animation; PD calculator |
| **Electromotive Force** (ε = W/Q) | EMF vs PD comparison; internal resistance simulator with live circuit |
| **Resistance & Ohm's Law** (R = V/I) | Live V–I graph with circuit diagram; wire length/thickness demo |
| **Non-Ohmic Conductors** | Animated I–V graphs for ohmic conductor, filament lamp & semiconductor diode |

### 📗 Chapter 2: DC Circuits
| Topic | Interactive Elements |
|---|---|
| **Series Circuits** | Live circuit simulator with electron flow; voltage drops update in real-time |
| **Parallel Circuits** | Branch current visualisation; electrons split at junctions |
| **Potential Divider** | Full divider simulator with voltage bar chart; thermistor & LDR sensor circuits |

### 📕 Chapter 3: Practical Electricity
| Topic | Interactive Elements |
|---|---|
| **Electrical Power** (P = VI) | Interactive power triangle (click to find P, V or I); power calculator |
| **Energy & Cost** (E = Pt) | Electricity bill calculator with kWh and cost |
| **Dangers & Safety** | Interactive 3-pin plug (click to learn); "Spot the Danger" room game; fuse rating calculator |

### 🎯 Quizzes
- 3 topic quizzes with **10 questions each**
- Instant feedback with detailed explanations
- Score tracking with emoji-based performance messages
- Questions shuffle on each attempt
- Retry functionality

## 🔧 Circuit Symbols

All circuit symbols follow the **Singapore/British convention**:

| Component | Symbol Style |
|---|---|
| Resistor | Rectangle (not zigzag) |
| Bulb/Lamp | Circle with cross (⊗) |
| Battery | Long & short parallel lines |
| Ammeter | Circle with "A" |
| Voltmeter | Circle with "V" |
| Switch | Dot-and-lever |
| Diode | Triangle with bar |
